## Reviewer 1: 技术可靠性

总体评价：稿件技术上有雄心，透明度较高，但核心 Bishan 结果目前由三个不同 regime 支撑：A 为 legacy research-side prepare，B/C 为 corrected package-side prepare，且 parcel/block 数、初始坡度和结果幅度均不同。这会显著削弱主结果的可解释性。

主要优点：县域尺度问题真实且较大；MSE 模型在 action ranking 上失败的诊断有价值；contrastive ranking loss 的作用机制讲得清楚；作者主动报告 restoration case 中 contrastive loss 几乎无收益，这是可信的负结果。

主要问题：Bishan 结果在 `-1.289%`、`-1.544%`、`-2.001%` 三个量级之间切换，且解释为“deployment-side variance”并不充分。由于准备流程修正导致初始 slope 从约 `8.41°` 到 `9.62°`，这不是普通随机误差，而是输入定义变化。

必须修改：建议选择一个唯一主分析 pipeline，最好是 corrected package-side prepare，然后在该 pipeline 上重跑 contrastive MPC、DRL baselines、OR baselines 和 GIS verification。若无法重跑，则应把 regime A 明确限定为 legacy comparison，把 B/C 作为 reproducibility/verification，不应混合作为同一 headline evidence。

建议姿态：Major revision。

## Reviewer 2: 科学重要性与 CE&E 适配性

总体评价：稿件更像一篇“可复现地理空间优化/AI planning 方法论文”，而不是已经充分证明环境科学结论的论文。对 CE&E 而言，应用价值有潜力，但需要把 proxy optimization 转化为更清楚的土地、生态或政策意义。

主要优点：聚焦山地丘陵破碎农田整治，是 CE&E 读者会关心的真实问题；工具链面向 ArcGIS/CLI，具有实际部署潜力；两县结果和公共 restoration case 扩展了适用边界讨论。

主要问题：坡度下降、连通性提升、百亩方数量变化是规划 proxy，但稿件没有充分说明这些变化对侵蚀风险、机械化可达性、产量、成本、政策合规或农户影响意味着什么。尤其 Bishan 中 qualifying baimu-fang area 下降数百公顷，作者称其为“intentional trade-off”，但需要更强政策解释。

必须修改：增加实际影响分析，例如绝对坡度变化、受影响面积、超过 6°/15° 等阈值的地块比例变化、乡镇分布公平性、百亩方面积损失是否触碰政策红线、不同 reward weights 下 trade-off 曲线。否则“food security and erosion control”应降调为潜在应用。

建议姿态：Major revision。

## Reviewer 3: 基线、泛化与可读性

总体评价：稿件材料丰富，但论证线过多：ranking failure、MPC vs OR、县域整治、restoration cross-domain、ArcGIS deployment 同时展开，导致主贡献边界不够锐利。

主要优点：作者试图区分“contrastive loss 何时有用”和“learned surrogate MPC 何时优于 OR”，这是比单纯报 SOTA 更成熟的写法。数据和代码可用性声明也较具体。

主要问题：基线证据仍偏弱。DRL baseline 是 in-house，run logs 不完全公开；OR baseline 的 matched wall-budget 设计合理但不充分；true-env MPC 对照因为只能用 200-action subset 和 random continuation，不能作为严格上界。`~30 ms/step` 阈值来自少量案例和 injected delay，当前不宜表述为强 criterion。

必须修改：补充更强 OR/local-search baseline，至少包括更长 wall-time budget、统一 corrected pipeline 下的 simulated annealing/GA/MILP 或 problem-specific heuristic；对 `H`、`K`、reward weights、ranking-loss margin 和 `lambda_rank` 做敏感性分析。将“two-quantity criterion”改为“empirical diagnostic / working hypothesis”。

建议姿态：Major revision。

## Cross-review Synthesis

这篇稿件最强的部分是：在真实县域地籍规划中发现 MSE surrogate 的 within-state ranking failure，并用 pairwise ranking loss 修复；同时提供可部署工具链和较完整验证。

最弱的部分是：主结果 evidence hierarchy 不够干净，环境科学影响仍主要停留在代理指标，泛化声明和 step-cost threshold 稍显过度。
